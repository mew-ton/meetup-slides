# JIS

2025/11/13

![bg 15%](./_assets/logos/jis.png)

---
## JIS とは？

- 日本工業規格 (Japanese Industrial Standards)
- 日本国内の工業製品やサービスの標準化を行う機関
- 工業標準調査会 (JISC) が運営

---

## JIS X 8341-3 とは？

- 日本のWebアクセシビリティに関する産業規格
- W3Cの国際規格WCAG（Web Content Accessibility Guidelines）を日本語化したもの
- 高齢者・障害者を含めて、誰もがWebサイトを利用できることを目指す

---

## JIS X 8341-3 のニュース

**改正原案作成委員会が発足**（11/13発表）
- 2025年10月30日に委員会を発足
- JIS X 8341-3の改正原案を作成
- 2026年にも改正される見込み

---

## 背景：規格改正の流れ

**時系列**
- 2023年10月: WCAG 2.2がW3Cから勧告
- 2025年9月: ISO/IEC 40500がWCAG 2.2ベースに改正
- 2025年10月: JIS X 8341-3改正委員会が発足 ← 今回
- 2026年予定: JIS X 8341-3が改正

**JISは国際規格（ISO）と内容を一致させる「一致規格」**

---

## WCAG 2.2で何が変わった？

**9つの達成基準を追加**
- フォーカスの可視性強化（他のコンテンツで隠されない）
- タッチ操作改善（24×24ピクセル以上）
- ヘルプ機能は常に同じ場所に
- 同一セッション内での情報再入力不要
- **パスワードのコピペ禁止はNG**

# TypeScript

2025/10/28

![bg 15%](./_assets/logos/typescript.png)

---

## TypeScript とは？

- JavaScriptに型システムを追加した言語
- Microsoftが開発、2012年にリリース
- コンパイル時にエラーを検出でき、大規模開発で広く使われる

---

## TypeScript のニュース

**GitHubで最も使用される言語に**（10/28発表）
- GitHub Octoverse 2025レポートより
- 2025年8月、PythonとJavaScriptを抜いて1位に
- 10年以上ぶりの大きな言語シフト
- 2025年だけで100万人以上の開発者が増加（+66%）

---

## なぜTypeScriptが人気？

**AI時代との相性**
- AI支援開発で型システムが重要視される
- 型情報があるとLLMがより正確なコードを生成
- ほぼすべての主要フロントエンドフレームワークがTypeScriptをデフォルトに

**開発体験の向上**
- エディタの自動補完・リファクタリング機能が強力
- 実行前にエラーを検出できる

---

# Node.js

2025/11/11

![bg 15%](./_assets/logos/nodejs.png)

---

## Node.js とは？

- サーバーサイドで動作するJavaScript実行環境
- V8エンジンを搭載（ChromeとおなじJSエンジン）
- 同様のJS実行環境としてDeno、Bun等あるが、利用率が最も高い

---

## Node.js のニュース

**v25.2.0リリース**（11/11）
- Type stripping（型削除）が安定版に昇格
- TypeScriptファイルを直接実行可能に

---

## Type stripping とは？

**従来の方法**
```bash
# TypeScriptをJavaScriptにコンパイルしてから実行
tsc index.ts    # index.js を生成
node index.js   # 実行
```

**Type stripping**
```bash
# TypeScriptを直接実行！
node index.ts
```

型情報だけを削除して即座に実行できる

---

## Type stripping の背景

**TypeScriptの人気上昇を受けて**
- DenoやBunは最初からTypeScript対応
- Node.jsも追従し、Type stripping機能を実装
- TypeScriptファーストの開発がより身近に
