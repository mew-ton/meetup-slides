## "ニューヨーク州" とは

- **回避不可** な警告を, 初回および定期的に表示させることを **義務化**

## "mquickjs" とは

- QEMU, FFmpeg, QuickJS の作者である Fabrice Bellard 氏による新作
- 組み込み向けの Micro QuickJS (ROM 100kB, RAM 10kB で動作)

- 組み込みシステム向けに最適化された超軽量 JS エンジン

## Tailwind CSS

2026/01/06

![bg 15%](./_assets/logos/tailwindcss.svg)

---

## "Tailwind CSS" とは

- Utility-first CSS framework (ユーティリティファーストな CSS フレームワーク)
- クラス名を組み合わせるだけでデザインを素早く構築可能
- 近年はデザインシステム構築の基盤としても広く採用されている

---

## Tailwind CSS のニュース

**公式ドキュメント最適化 PR が拒否され話題に**

- ドキュメントを LLM に最適化する `/llms.txt` 提案 PR が拒否された
- 作者が **「売上 80% 減 / トラフィック 40% 減」** と激白
- すでに **従業員の 75% をレイオフ** しており、生存をかけた判断
- OSS ビジネスモデルと AI の関係性について議論を呼んでいる

---

## Tailwind Labs の収益モデル

- 本体は OSS (無料) で利用者を拡大
- 収益源は **Tailwind UI** (有料コンポーネント集) の販売
- ドキュメント閲覧からの有料製品購入が主要な導線となっている

---

## OSS ビジネスの曲がり角

- **「ドキュメントで集客し、有料版で稼ぐ」** という従来モデルが AI により機能不全に
- ユーザーは便利になるが、開発元の収益源が絶たれるジレンマ
- 今後 OSS がどのように持続可能性を確保するかが問われている

## jQuery とは

- 2006 年に登場した、世界で最も使われている JavaScript ライブラリ (**今年で 20 周年** !)
- "Write Less, Do More" を掲げ、DOM 操作や Ajax を簡素化した
- 現代の Web 標準 API の多くは jQuery の影響を受けている

---

## jQuery 4.0.0 のニュース

**jQuery 4.0.0 がリリース**

- IE 10 以下のサポートを終了 (IE 11 以降のみサポート)
- 多くの非推奨 API を削除し、コードベースをスリム化
- これからもメンテナンスは継続される (Still relevant)

## Astro

2026/01/16

![bg 15%](./_assets/logos/astro.svg)

---

## "Astro" とは

- パフォーマンス重視の静的サイトジェネレータ
- "Islands Architecture" を採用し、必要な部分だけ JS をロードする
- React, Vue, Svelte など多様なフレームワークを併用可能

---

## Astro のニュース

**Cloudflare が Astro を買収**

- Astro は Cloudflare の一部となるが、OSS (MIT License) として継続
- 開発チームは Cloudflare に合流しフルタイムで開発を継続
- エッジコンピューティングとの親和性がさらに高まると期待される
